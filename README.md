# ProConnect — Landing Page Prototype

First-visit landing page for **ProConnect**, the ASEAN-wide hospitality hiring platform built with ASEANTA and the Indonesian Ministry of Tourism.

## What's inside
- Single self-contained `index.html` (Tailwind via CDN, Inter font, Lucide-style inline SVGs)
- Combines Features, Pricing, Browse Jobs, About Us, and Contact into one conversion-optimized page
- Primary CTAs: **Request Demo** and **Sign Up Free**

## Local preview
Just open `index.html` in any browser — no build step.

## Deploy
Auto-deploys to Netlify on push to `main`. See `netlify.toml` for config.
